## 📱 Overview

**Cosmetics** is a Flutter-based mobile application designed to showcase beauty and skincare products with a modern, elegant, and user-friendly interface.

The app provides a smooth browsing experience for users to explore beauty products, view product details, and manage authentication through login and sign-up pages.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🌷 **Elegant UI** | Clean and modern design tailored for beauty products |
| 🔐 **Authentication** | Login and Sign-Up pages |
| 🛍️ **Product Details** | Detailed view for each product |
| 🛒 **Shopping Cart** | Simple and intuitive cart management |
| 🔎 **Search** | Easily find products by name or category |

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed → [flutter.dev](https://flutter.dev/docs/get-started/install)
- Android Studio or VS Code
- A connected device or emulator

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/DohaNoamanEssa/cosmetics-app.git

# 2. Navigate to the project folder
cd cosmetics-app

# 3. Install dependencies
flutter pub get

# 4. Run the app
flutter run
```
